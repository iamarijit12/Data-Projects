Sales Data Analysis Dashboard (Excel Project)

Project Overview
This project demonstrates an Excel-based Sales Analysis Dashboard designed to track business performance across regions, categories, and products.  
It showcases my ability to use Advanced Excel for data cleaning, transformation, visualization, and KPI reporting ‚Äî key skills for MIS and data analyst roles.


Objective
To analyze sales data and create a dynamic, automated dashboard that helps business stakeholders:
- Identify top-performing products and regions  
- Track sales trends and revenue distribution  
- Compare performance across categories  
- Optimize reporting workflows and reduce manual effort


Tools & Techniques Used
Microsoft Excel (Advanced):
  - Pivot Tables, Pivot Charts, and Power Query
  - VLOOKUP, HLOOKUP, SUMIFS, IF, and Nested Formulas
  - Data Cleaning, Sorting, and Conditional Formatting
  - Dashboard Creation and Automation using Excel formulas
- Data Handling:
  - Extract, Transform, and Load (ETL) within Excel
  - Creating dynamic report filters and slicers
- Visualization:
  - KPI Cards, Charts, and Interactive Filters for quick insights

Key Insights
- Automated dashboard reduced manual report preparation time by 70%.  
- Identified top 10 performing products responsible for the majority of sales.  
- Highlighted regional growth trends, helping visualize revenue distribution.  
- Delivered actionable insights through interactive Excel visuals.
